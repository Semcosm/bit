# Type System

当前原型只实现了一个极小的静态类型系统，重点是先把名字解析、
表达式检查和 LLVM IR lowering 串起来。

## 当前可用类型

- `i32`

## 当前规则

- 函数参数必须显式写出类型
- 函数返回类型必须显式写出类型
- 局部绑定必须显式写出类型
- `let` 初始化表达式的类型必须和声明类型一致
- `return` 表达式的类型必须和函数返回类型一致
- 函数调用会先按函数名解析，再检查每个实参类型
- 函数调用的实参数量必须与形参数量一致
- 一元负号当前只接受 `i32`
- 二元 `+`、`-`、`*`、`/` 当前都只接受 `i32`
- 十进制整数字面量默认按 `i32` 检查
- 允许通过 `-2147483648` 表达 `i32` 最小值

## 当前限制

- 模块内允许多个函数，但必须定义零参数 `main`
- `main` 必须返回 `i32`
- 没有隐式类型转换
- 没有布尔、比较、块作用域或复合类型
